# Portfolio Website

## Project Title
**Portfolio Website — Md. Hasmir Hassan**

## Project Description
A concise, responsive portfolio website that showcases projects, skills, and contact information. Built with accessibility and performance in mind, it includes a mobile-friendly layout, interactive elements (smooth scrolling, mobile navigation), and a contact form with client-side validation.

## Technologies Used
- HTML5
- CSS3 (responsive design)
- JavaScript (vanilla)
- Font Awesome (icons)
- Optional: Netlify / Vercel for deployment

## Folder Structure
```
portfolio/
├── index.html        # Main HTML file (site markup)
├── styles.css        # Styling and responsive rules
├── script.js         # JavaScript for interactivity
├── MyCV.jpg          # CV file (view/download)
├── profile-photo.jpg # Profile image used in hero
└── README.md         # Project documentation
```
